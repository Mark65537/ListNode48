# Описание задачи
Для типа «односвязанный неизменяемый список», определенного как </p>
  Class ListNode<T> {</p>
      Public readonly T Value;</p>
      Public readonly ListNode<T> Next;</p>
      …</p>
  }</p>
Написать две функции: замены элемента в списке и объединения двух списков.
